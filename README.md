## 🖥️ What this is
Imgui_tabs is a tab module for ImGui. It helps you add tabbed panels to apps and tools that use ImGui. Tabs make it easier to group content and switch between views in one window.

This module is built to stay useful across future ImGui updates. That makes it a good fit if you want a tab system that is easy to keep in place.

## ✨ What you can do
- Add tab bars to your ImGui app
- Group settings, tools, or data views
- Switch between panels without opening new windows
- Keep your interface tidy
- Use a module that should stay compatible with later ImGui changes

## 🔧 How to use
Imgui_tabs is meant to be used inside an ImGui project. After you download it, place the module files where your project can reach them.

Typical use looks like this:

- Copy the tab module files into your project
- Include the needed header or source file
- Call the tab functions where you build your UI
- Add labels for each tab
- Put your content inside each tab area

A simple layout may include:

- Settings
- Logs

That structure helps users move through the app without confusion.
